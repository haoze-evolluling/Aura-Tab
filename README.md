# Aura Tab - 现代化浏览器新标签页

一个简洁、现代化的浏览器新标签页，具备玻璃态质感设计和丰富的功能。

## ✨ 特性

- 🎨 **玻璃态设计** - 现代化的毛玻璃效果界面
- ⏰ **实时时钟** - 显示当前时间和日期，支持时区信息
- 🔍 **智能搜索** - 支持搜索建议和多搜索引擎
- 🔗 **快捷访问** - 自定义常用网站快捷方式
- 📱 **全面响应式设计** - 完美适配桌面、平板、手机等各种设备屏幕
- ⚡ **性能优化** - 自适应性能调整
- 🎯 **分布式架构** - 模块化代码结构

## 🚀 功能介绍

### 时钟组件
- 实时显示当前时间和日期
- 点击时钟查看详细时间信息
- 支持时区显示和UTC时间
- 智能问候语功能

### 搜索功能
- 智能搜索建议
- 支持直接URL访问
- 多搜索引擎支持（Google、Bing、百度等）
- 键盘快捷键支持（Ctrl+K）

### 快捷访问
- 自定义添加常用网站
- 自动获取网站图标
- 支持编辑和删除
- 右键上下文菜单
- 默认预设常用网站

### 视觉效果
- 动态渐变背景
- 玻璃态毛玻璃效果
- 流畅的动画过渡
- 全面响应式布局
- 多种光晕和阴影效果
- 支持触摸设备优化

### 响应式特性
- **桌面端** (1024px+) - 4-5列快捷方式网格，完整功能展示
- **平板端** (768px-1023px) - 3列网格布局，优化的间距和字体
- **手机端** (320px-767px) - 2列网格布局，触摸友好的按钮尺寸
- **横屏模式** - 特别优化的紧凑布局
- **触摸设备** - 增大点击区域，优化交互体验
- **高分辨率屏幕** - 字体渲染优化
- **无障碍支持** - 减少动画模式支持

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代CSS特性，包括backdrop-filter、grid、flexbox
- **JavaScript ES6+** - 模块化开发，类和异步编程
- **Web APIs** - localStorage、Intersection Observer等

## 🎮 使用方法

1. **搜索**
   - 在搜索框中输入关键词或网址
   - 使用 `Ctrl+K` 快速聚焦搜索框
   - 支持键盘导航选择建议

2. **添加快捷方式**
   - 点击"添加快捷方式"按钮
   - 输入网站名称和地址
   - 使用 `Ctrl+Shift+A` 快速添加

3. **管理快捷方式**
   - 右键点击快捷方式查看菜单
   - 支持编辑、删除、复制链接
   - 悬停显示删除按钮

4. **时钟功能**
   - 点击时钟查看详细时间信息
   - 自动显示时区和UTC时间

## ⚙️ 配置选项

应用支持以下本地存储配置：

- `quickAccessShortcuts` - 快捷访问网站列表
- `searchHistory` - 搜索历史记录
- `hasVisited` - 首次访问标记
- `appVersion` - 应用版本信息

## 📱 设备支持

### 完美适配
- 🖥️ **桌面设备** - 1024px及以上，支持鼠标悬停效果
- 📱 **平板设备** - 768px-1023px，优化的触摸交互
- 📱 **手机设备** - 320px-767px，专为移动端优化
- 🔄 **横屏模式** - 自动调整布局适应横屏显示

### 响应式断点
| 设备类型 | 屏幕宽度 | 快捷方式列数 | 特殊优化 |
|---------|---------|------------|---------|
| 大屏桌面 | ≥1400px | 5列 | 最大显示空间 |
| 标准桌面 | 1024-1399px | 4列 | 默认体验 |
| 平板 | 768-1023px | 3列 | 触摸优化 |
| 大屏手机 | 640-767px | 3列 | 移动优化 |
| 标准手机 | 480-639px | 2列 | 紧凑布局 |
| 小屏手机 | 320-479px | 2列 | 超紧凑 |

### 触摸设备优化
- ✅ 增大可点击区域（最小44px）
- ✅ 禁用不必要的悬停效果
- ✅ 优化的触摸反馈动画
- ✅ 删除按钮始终可见

## 🔧 开发说明

### 本地开发
1. 克隆项目到本地
2. 使用本地服务器打开 `index.html`

- 在 `js/` 目录下添加新的组件模块
- 在 `main.js` 中注册新组件
