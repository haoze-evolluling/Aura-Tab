# Aura Tab - 现代化浏览器新标签页

一个简洁、现代化的浏览器新标签页，具备玻璃态质感设计和丰富的功能。

## ✨ 特性

- 🎨 **玻璃态设计** - 现代化的毛玻璃效果界面
- ⏰ **实时时钟** - 显示当前时间和日期，支持时区信息
- 🔍 **智能搜索** - 支持搜索建议和多搜索引擎
- 🔗 **快捷访问** - 自定义常用网站快捷方式
- ✨ **粒子动画** - 动态背景粒子效果
- 📱 **响应式设计** - 完美适配各种设备屏幕
- ⚡ **性能优化** - 自适应性能调整
- 🎯 **分布式架构** - 模块化代码结构

## 🚀 功能介绍

### 时钟组件
- 实时显示当前时间和日期
- 点击时钟查看详细时间信息
- 支持时区显示和UTC时间
- 智能问候语功能

### 搜索功能
- 智能搜索建议
- 支持直接URL访问
- 多搜索引擎支持（Google、Bing、百度等）
- 键盘快捷键支持（Ctrl+K）

### 快捷访问
- 自定义添加常用网站
- 自动获取网站图标
- 支持编辑和删除
- 右键上下文菜单
- 默认预设常用网站

### 视觉效果
- 动态渐变背景
- 交互式粒子动画
- 玻璃态毛玻璃效果
- 流畅的动画过渡
- 响应式布局

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代CSS特性，包括backdrop-filter、grid、flexbox
- **JavaScript ES6+** - 模块化开发，类和异步编程
- **Web APIs** - localStorage、Intersection Observer等

## 🎮 使用方法

1. **搜索**
   - 在搜索框中输入关键词或网址
   - 使用 `Ctrl+K` 快速聚焦搜索框
   - 支持键盘导航选择建议

2. **添加快捷方式**
   - 点击"添加快捷方式"按钮
   - 输入网站名称和地址
   - 使用 `Ctrl+Shift+A` 快速添加

3. **管理快捷方式**
   - 右键点击快捷方式查看菜单
   - 支持编辑、删除、复制链接
   - 悬停显示删除按钮

4. **时钟功能**
   - 点击时钟查看详细时间信息
   - 自动显示时区和UTC时间

## ⚙️ 配置选项

应用支持以下本地存储配置：

- `quickAccessShortcuts` - 快捷访问网站列表
- `searchHistory` - 搜索历史记录
- `hasVisited` - 首次访问标记
- `appVersion` - 应用版本信息

## 🎨 自定义样式

项目使用CSS自定义属性，可以轻松自定义主题：

```css
:root {
    --primary-color: #667eea;      /* 主色调 */
    --secondary-color: #764ba2;    /* 辅助色 */
    --accent-color: #f093fb;       /* 强调色 */
    --text-primary: #ffffff;       /* 主文字色 */
    --glass-bg: rgba(255, 255, 255, 0.1);  /* 玻璃背景 */
}
```

## 🔧 开发说明

### 本地开发
1. 克隆项目到本地
2. 使用本地服务器打开 `index.html`
3. 或直接在浏览器中打开文件

### 代码结构
- 采用模块化设计，每个功能独立封装
- 使用ES6类和现代JavaScript特性
- 响应式设计，支持移动端
- 性能优化，包括防抖、节流、懒加载

### 自定义开发
- 在 `js/` 目录下添加新的组件模块
- 在 `main.js` 中注册新组件
- 使用统一的样式规范和命名约定
